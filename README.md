# Persona Companion Images

Image assets for the [Persona Companion](https://github.com/Sentovibes/Persona-Companion-App) Android app.

## Contents

This repository hosts high-quality images for:
- **Personas** from Persona 3, 4, 5 and their variants (FES, Portable, Golden, Royal, Reload, Episode Aigis)
- **Enemies** from various Persona games including bosses and mini-bosses

## Download

Images are distributed as a single ZIP file via GitHub Releases:
- **Latest Release**: [v1.0](https://github.com/Sentovibes/persona-companion-images/releases/tag/v1.0)
- **File Size**: ~1.31 GB
- **Format**: PNG images

## Usage

These images are automatically downloaded by the Persona Companion app when users first launch the app (release builds only). Debug builds include images bundled in the APK.
